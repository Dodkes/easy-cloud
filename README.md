# Easy cloud storage

Provides nice UI/UX to store/backup your data to remote device.

## Overview

![./docs/flow.svg](./docs/flow.svg)

## Installation

### Client:

```sh
npm i
```

```sh
npm run dev
```

### Server:

```sh
nodemon server.js
```

To reset storage (delete all):

```sh
npm run storage:clear
```
